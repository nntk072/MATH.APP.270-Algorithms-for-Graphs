# MATH.APP.270-Algorithms-for-Graphs
This is Tampere University course, for learning and modifying algorithms for graph. 
All information can be found in https://sis-tuni.funidata.fi/student/courseunit/otm-36a14850-e522-4246-8244-87f5f07f0ab9/brochure

Contents of each "code" week:

Week 3: Applying modified BFS algorithm for finding the shortest route and the number of paths of that route

Week 5: Applying Ford-fulkerson algorithm for finding the flow which is as large as possible

Week 7: Applying Page-rank algorithm for modifying and fixing the "pass-the-paton" game
